# Hytale Mod/Plugin Template

Features:
- Gradle.
- Custom setup and run tasks.
- (unfinished for now) CI.

## Instructions
1. Fork this repository.
2. Open it in your IDE of choice.
3. Build the project.
4. On sync follow instructions in the terminal (you'll need to log into your account to allow `hytale-downloader` to download server files).
5. Voila!
