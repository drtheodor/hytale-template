# Hytale Mod/Plugin Template

Features:
- Gradle.
- Custom setup and run tasks.
- (unfinished for now) CI.

## Instructions
1. Fork this repository.
2. Open it in your IDE of choice.
3. Run `setupHytaleServer` Gradle task (& refresh the project, if needed).
4. On reload follow instructions in the build (sync) menu (you'll need to log into your account to allow `hytale-downloader` to download server files).
5. Rename the project in `settings.gradle`, update the group in `gradle.properties`, etc. Enjoy.
